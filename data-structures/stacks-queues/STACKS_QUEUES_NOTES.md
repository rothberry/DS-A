# Stacks and Queues

## Stacks

Last In First Out (LIFO)
Ex: A stack of plates, where you can only grab the top of the Stack

Lookup => O(n)
pop => O(1)

## Queues

First In First Out = (FIFO)

Rollercoaster line

Lookup    =>  O(n)
Enqueue   =>  O(1)  (Push)
Dequeue   =>  O(1)  (Pop)
Peek      =>  O(1)
